# DealClaw Demo

这是一个智能客户画像与商机匹配系统的演示项目。

## 快速试用

按照以下步骤体验 Demo：

1. **输入业务描述** → 填写「我们是做户外用品的」
2. **选择出口市场** → 选择「主要出口美国」
3. **确认客户画像** → 查看生成的画像并确认
4. **输入获客需求** → 填写「帮我找批发商」
5. **设置采购规模** → 选择「年采购100-500万」
6. **启动任务** → 确认后系统将开始匹配商机

## 启动项目

```bash
npm install
npm run dev
```

访问 http://localhost:8080 开始试用。

## 技术栈

- React + TypeScript + Vite
- Tailwind CSS
- shadcn/ui
