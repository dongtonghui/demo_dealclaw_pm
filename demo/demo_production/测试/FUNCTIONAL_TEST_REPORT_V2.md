## 🎬 功能演示流程

### 流程1: Onboarding + 创建任务

```
1. 打开首页 → 显示主管Agent欢迎语
2. 输入"我们是做户外用品的" → Agent追问市场
3. 输入"主要出口美国" → 生成企业画像卡片
4. 点击"确认无误" → 引导输入获客目标
5. 输入"帮我找美国户外用品批发商" → 追问细节
6. 输入"年采购100-500万美元，3个月内" → 生成客户画像
7. 点击"确认无误" → 显示协调Agent动画
8. 生成获客方案卡片 → 显示SEO+Email策略
9. 点击"确认执行" → 任务启动
```

### 流程2: 查询进度

```
1. 输入"任务进展如何" → 显示任务进度看板
2. 查看SEO进度 → 6篇文章、3个关键词排名
3. 查看Email进度 → 450封发送、96%送达率
4. 查看线索收获 → 23条线索、8条高意向
```

### 流程3: 查看线索

```
1. 输入"查看线索" → 显示线索收件箱卡片
2. 显示3条高意向线索 → Summit Camping (92分)
3. 点击线索 → 展开详情页
4. 查看互动历史 → 邮件发送→打开→网站访问→回复
5. 点击"生成回复" → 生成个性化回复建议
```

### 流程4: SEO文章生成

```
1. 输入"@SEO Agent 生成文章" → SEO Agent响应
2. 显示SEO策略卡片 → 4个关键词、4周内容计划
3. 点击"查看首篇文章" → 显示文章预览
4. 查看质量检测 → 原创度96%、可读性良好
5. 点击"编辑" → 进入编辑模式
6. 点击"发布" → 发布成功确认
```

### 流程5: 邮件预览

```
1. 输入"@Email Agent 查看客户" → 显示客户列表
2. 显示856家匹配客户 → 5家高意向推荐
3. 点击"预览邮件" → 显示邮件预览卡片
4. 查看收件箱模拟 → Gmail样式预览
5. 查看个性化亮点 → 3个个性化点标注
6. 查看专业度评分 → 92/100
```

### 流程6: 数据看板

```
1. 输入"本周效果如何" → 显示数据看板
2. 查看核心指标 → 线索+27%、高意向+43%
3. 查看来源分布 → Email 68%、SEO 25%
4. 查看ROI → 4.4x回报
5. 查看优化建议 → 2条智能建议
```
