# SEO Agent 快速测试清单

> 适用于手动验证所有SEO功能

---

## 🚀 快速测试流程（10分钟）

### 第一步：关键词推荐 (2分钟)
```
输入: @SEO Agent 推荐关键词
```
- [ ] 显示关键词卡片
- [ ] 点击"高机会词"筛选
- [ ] 搜索框输入 "camping"
- [ ] 勾选2个关键词，点击"添加到策略"

### 第二步：文章编辑 (2分钟)
```
输入: @SEO Agent 编辑文章
```
- [ ] 显示文章编辑器
- [ ] 切换到"SEO优化"标签
- [ ] 查看SEO得分
- [ ] 切换到"预览效果"标签
- [ ] 点击"保存文章"

### 第三步：站点连接 (2分钟)
```
输入: @SEO Agent 绑定站点
```
- [ ] 点击"添加站点"
- [ ] 选择WordPress
- [ ] 输入测试信息：
  - 名称: Test Site
  - URL: https://test.com
  - API Key: test123
- [ ] 点击"连接站点"

### 第四步：文章数据 (1分钟)
```
输入: @SEO Agent 文章数据
```
- [ ] 显示数据分析卡片
- [ ] 点击"30天"切换时间范围
- [ ] 点击第二篇文章查看数据

### 第五步：竞品分析 (1分钟)
```
输入: @SEO Agent 竞品分析
```
- [ ] 显示竞品分析卡片
- [ ] 切换到"关键词差距"标签
- [ ] 切换到"策略建议"标签

### 第六步：一键建站 (1分钟)
```
输入: @SEO Agent 生成站点
```
- [ ] 显示建站向导
- [ ] 选择"极简风格"模板
- [ ] 选择"自然绿"配色
- [ ] 输入站点名称
- [ ] 点击"下一步"
- [ ] 点击"开始生成"
- [ ] 等待生成完成

### 第七步：健康检查 (1分钟)
```
输入: @SEO Agent 健康检查
```
- [ ] 显示健康检查卡片
- [ ] 查看总体评分
- [ ] 点击一个有问题的检查项展开详情
- [ ] 点击"重新扫描"

## 🔧 边界测试

### 空状态测试
- [ ] 未添加站点时，发布文章按钮应提示先连接站点
- [ ] 未生成文章时，编辑文章应创建新文章

### 错误处理测试
- [ ] 输入无效URL时，站点连接应失败
- [ ] 文章标题为空时，保存应提示必填

### 移动端测试
- [ ] 在移动设备上打开所有卡片
- [ ] 确认布局自适应
- [ ] 确认按钮可点击
