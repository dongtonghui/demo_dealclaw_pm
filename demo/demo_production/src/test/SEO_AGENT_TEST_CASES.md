## 1. 关键词推荐功能 (SEO-001/002)

### TC-001: 关键词推荐基础功能
**前置条件**: 已完成企业画像设置
**测试步骤**:
1. 在聊天框输入 `@SEO Agent 推荐关键词`
2. 等待Agent响应

**预期结果**:
- 显示关键词推荐卡片
- 展示8-10个相关关键词
- 每个关键词包含：搜索量、难度、CPC、意图类型、竞争度、趋势
- 显示关键词相关变体

### TC-002: 关键词筛选功能
**前置条件**: 已显示关键词推荐卡片
**测试步骤**:
1. 点击"全部推荐" → "高机会词" 标签
2. 点击"全部推荐" → "热门趋势" 标签
3. 在搜索框输入 "wholesale"

**预期结果**:
- 高机会词：只显示难度为"easy"且竞争度为"low"的词
- 热门趋势：只显示趋势为"up"的词
- 搜索筛选：只显示包含"wholesale"的关键词

### TC-003: 关键词导出
**前置条件**: 已显示关键词推荐卡片
**测试步骤**:
1. 勾选多个关键词（点击复选框）
2. 点击"导出关键词"按钮

**预期结果**:
- 显示导出成功提示
- 关键词列表已导出

### TC-004: 添加到策略
**前置条件**: 已显示关键词推荐卡片
**测试步骤**:
1. 勾选3-5个关键词
2. 点击"添加到策略"按钮

**预期结果**:
- 显示确认消息"关键词已添加到您的SEO策略"
- Agent提示会根据这些关键词规划内容创作

### TC-009: 预览效果

## 3. 站点连接管理 (SEO-005)

### TC-010: 添加WordPress站点
**前置条件**: 无
**测试步骤**:
1. 输入 `@SEO Agent 绑定站点`
2. 点击"添加站点"按钮
3. 选择"WordPress"平台
4. 输入站点名称、URL、API密钥
5. 勾选"启用自动发布"

**预期结果**:
- 显示连接成功提示
- 站点列表显示新添加的WordPress站点
- 显示"已启用自动发布"标签
